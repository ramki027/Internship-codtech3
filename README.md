#  Spam Email Classifier (Machine Learning)

##  Overview
This project implements a machine learning model to classify messages as spam or ham using Scikit-learn.

---

##  Features
- Text preprocessing
- Feature extraction using CountVectorizer
- Model training using Naive Bayes
- Performance evaluation

---

##  Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
